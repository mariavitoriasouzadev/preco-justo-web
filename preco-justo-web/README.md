# Preço Justo

## Sobre o Projeto  

O **Preço Justo** é uma plataforma digital desenvolvida com o objetivo de reduzir a assimetria informacional enfrentada por pequenos produtores rurais. O projeto disponibiliza uma página web responsiva e um chatbot via WhatsApp, permitindo que agricultores consultem os preços de produtos agrícolas praticados em mercados e cooperativas locais.  

Dessa forma, busca-se:  
- Fortalecer a autonomia econômica dos produtores;  
- Permitir negociações mais justas;  
- Reduzir a dependência de intermediários.  

## Tecnologias Utilizadas  

- **Front-End:** ReactJS, JavaScript e Bootstrap 
- **Chatbot:** Power Apps  
- **Integração de Dados:** APIs  
- **Controle de Versão:** Git e GitHub

## Sobre as Tecnologias

- **Vite:** Ferramenta moderna para desenvolvimento front-end, que oferece inicialização rápida e recarregamento instantâneo durante o desenvolvimento.
- **Bootstrap:** Framework CSS que facilita a criação de interfaces responsivas e componentes visuais prontos para uso.
- **React:** Biblioteca JavaScript para construção de interfaces de usuário dinâmicas e reativas.

## Como rodar o projeto localmente

1. **Pré-requisitos**
   - Node.js versão **22.12.0** ou superior (recomendado)
   - [nvm](https://github.com/coreybutler/nvm-windows) para gerenciar versões do Node (opcional, mas recomendado)

2. **Instalação**
   ```bash
   git clone https://github.com/mariavitoriasouzadev/preco-justo-web.git
   cd preco-justo-web/preco-justo-web
   npm install
   ```

3. **Rodando a aplicação**
   ```bash
   npm run dev
   ```
   Acesse [http://localhost:5173](http://localhost:5173) no navegador para visualizar o projeto.
